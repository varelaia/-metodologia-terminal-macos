======================================================================
     MANUAL INTEGRADO DE TERMINAL MACOS (USUARIO, TÉCNICO Y OPERATIVO)
======================================================================
Este manual consolida todas las herramientas, configuraciones y scripts 
implementados en esta sesión en tu Mac.

======================================================================
                     PARTE 1: MANUAL DE USUARIO
======================================================================
Diseñado para el uso diario de la terminal de forma sencilla y eficiente.

1. Navegación Automática
   * Al abrir tu terminal física en tu Mac, se abrirá por defecto 
     directamente en tu carpeta de proyectos: `/Users/yarelyvarela/Code`.
   * Tienes un acceso directo (enlace simbólico) llamado "Code" en tu 
     Escritorio para entrar rápido usando el Finder.

2. Herramientas de Productividad Instaladas:
   * Navegación Veloz (zoxide):
     En lugar de escribir rutas largas con "cd", escribe "z" seguido de 
     una palabra clave de la carpeta a la que quieres ir.
     Ejemplo: "z metodologia" te llevará instantáneamente al repositorio.
   * Buscador de Historial (fzf):
     Presiona la combinación de teclas "Ctrl + R" en tu terminal para 
     abrir un buscador interactivo en vivo de todos tus comandos antiguos.
   * Visualización de Archivos (bat):
     Escribe "bat [nombre-archivo]" para ver su contenido con colores, 
     números de línea y soporte de Git.
   * Listar Directorios con Colores (eza):
     Escribe "eza -l --grid" para listar carpetas con iconos y colores.
   * Monitor del Sistema (btop):
     Escribe "btop" para abrir una interfaz visual hermosa en tu consola 
     que te muestra el uso de CPU, RAM, discos y red en tiempo real.
   * Manuales Rápidos (tldr):
     Si olvidas cómo usar un comando, escribe "tldr [comando]" (ej: "tldr git") 
     para ver ejemplos prácticos resumidos en una sola pantalla.

3. Ejecución de Scripts de Automatización:
   * Para optimizar tu Mac de forma segura:
     cd /Users/yarelyvarela/Code
     ./optimizar_equipo.sh
   * Para aprovisionar o reinstalar tu entorno en otra Mac limpia:
     cd /Users/yarelyvarela/Code
     ./instalar_entorno.sh

======================================================================
                     PARTE 2: MANUAL TÉCNICO
======================================================================
Destinado a administradores de sistema o agentes para comprender la arquitectura.

1. Archivos de Configuración de la Shell (~/.zshrc)
   Las inicializaciones y rutas de las nuevas herramientas están enlazadas 
   al final del archivo de configuración global de Zsh:
   * PATH de Python 3.12:
     export PATH="/usr/local/opt/python@3.12/libexec/bin:$PATH"
   * Inicialización de Zoxide:
     eval "$(zoxide init zsh)"
   * Integración de Fzf:
     source <(fzf --zsh)
   * Carga de Plugins de Consola:
     source /usr/local/share/zsh-autosuggestions/zsh-autosuggestions.zsh
     source /usr/local/share/zsh-syntax-highlighting/zsh-syntax-highlighting.zsh

2. Parámetros del Sistema macOS Modificados (Defaults)
   Se aplicaron los siguientes defaults de sistema en macOS:
   * defaults write com.apple.finder AppleShowAllFiles -bool true (Ver archivos ocultos)
   * defaults write NSGlobalDomain AppleShowAllExtensions -bool true (Ver extensiones)
   * defaults write NSGlobalDomain KeyRepeat -int 1 (Velocidad de repetición de teclas al máximo)
   * defaults write NSGlobalDomain InitialKeyRepeat -int 10 (Retraso mínimo de repetición)
   * defaults write NSGlobalDomain NSAutomaticSpellingCorrectionEnabled -bool false (Desactivar autocorrección global)

3. Criptografía y Conexiones Git
   * Llave SSH generada: Algoritmo Ed25519 (~/.ssh/id_ed25519).
   * Mapeo de Red SSH: Archivo ~/.ssh/config configurado para mapear 
     github.com asociando la llave id_ed25519 de forma permanente al Llavero.

======================================================================
                     PARTE 3: MANUAL OPERATIVO
======================================================================
Define las tareas de mantenimiento recurrente para asegurar la salud de la Mac.

1. Mantenimiento de Software (Homebrew):
   * Para actualizar el índice local de fórmulas:
     brew update
   * Para actualizar e instalar parches de seguridad de todos tus paquetes:
     brew upgrade

2. Optimización Semanal del Sistema:
   Se recomienda ejecutar una vez a la semana el script "./optimizar_equipo.sh".
   Este script ejecuta un Premortem Gate en vivo comprobando si existen 
   servidores de bases de datos o desarrollo activos (puertos 3000, 5173, etc.) 
   antes de limpiar cachés de desarrollo para evitar pérdidas de estado.

3. Sincronización de Metodologías y Skills:
   La metodología sanitizada se encuentra publicada en el repositorio:
   https://github.com/varelaia/-metodologia-terminal-macos
   * Para subir actualizaciones desde la carpeta local:
     git add .
     git commit -m "Descripción del cambio"
     git push origin main
   * Para descargar cambios en otro equipo:
     git pull origin main
